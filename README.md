# TP-SALAIRE-EMPLOYE
Application web permettant de gérer le salaire des employés d'une entreprise

# Fonctionnalités
  - Ajouter des nouveaux employés
  - Supprimer des employés
  - modifier des employés
  - Calcul du salaire mensuel des employés

# Technologies
  -  FrontEnd ( Template Bootstrap )
  -  BackEnd ( Laravel 10 )
  -  BD ( MySQL )
